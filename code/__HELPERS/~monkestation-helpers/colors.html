/proc/is_color_dark_with_saturation Given a color in the format of "#RRGGBB", will return if the color is dark. Value is mixed with Saturation and Brightness from HSV.
/proc/is_color_dark_without_saturation it checks if a color is dark, but without saturation value. This uses Brightness only, without Saturation from HSV
/proc/get_color_brightness_from_hex returns HSV brightness 0 to 100 by color hex
