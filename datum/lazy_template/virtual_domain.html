virtual_domain
/datum/lazy_template/virtual_domain

Your safehouse here /datum/map_template/safehouse/your_type filename = "your_map.dmm"

Virtual Domains

This loads a base level, then users can select the preset upon it. Create your own: Read the readme file in the '_maps/virtual_domains' folder.
