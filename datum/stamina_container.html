Procs

adjustAdjust stamina by an amount.
nullspace_move_checkSignal handler for COMSIG_MOVABLE_MOVED to ensure that update_process() gets called whenever moving to/from nullspace.
pausePause stamina regeneration for some period of time. Will override previous cooldown if one was set. Will NOT override stop(), if stop() was used to halt stamina regen.
resumeResume stamina regeneration
revitalizeRevitalize the stamina to the maximum this container can have.

should_processReturns if the container should currently be processing or not.
stopStops stamina regeneration entirely until manually resumed.
update_processChecks to see if the container should be processing, and starts/stops it.
