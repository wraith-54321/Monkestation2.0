Vars

descThe description of the cassette.

nameThe name of the cassette.
statusThe status of this cassette.

Procs

copyCreate a copy of this cassette.

get_sideSimple helper to get a side of the cassette. TRUE is front side, FALSE is back side.

save_to_fileSaves the cassette to the data folder, in JSON format.
