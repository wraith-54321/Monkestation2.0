sticker
/obj/item/sticker

parent type for all other stickers. do not spawn directly

Vars

attachedThe thing we are attached to
contrabandIf the sticker should be disincluded from normal sticker boxes.
icon_statesA list of icon_states to pick an icon_state on Initialize, provided it is not null.
signal_turfThe turf our COMSIG_TURF_EXPOSE is registered to, so we can unregister it later.
sticker_overlayThe overlay we apply to things we stick to
throw_stick_rateThe prob() rate of the sticker sticking if its thrown.

Procs

attempt_attachAttempts to attach sticker to an object. Returns FALSE if atom has more than MAX_STICKER_COUNT stickers, TRUE otherwise. If no px or py were passed picks random coordinates based on a target's icon.
on_attached_qdelSignal handler for COMSIG_QDELETING, deletes this sticker if the attached object is deleted
on_igniteSignal handler for COMSIG_LIVING_IGNITED, deletes this sticker, if it is flammable
on_turf_exposeSignal handler for COMSIG_TURF_EXPOSE, deletes this sticker if the temperature is above 100C and it is flammable
peelMakes this sticker move from nullspace and cut the overlay from the object it is attached to, silent for no visible message.
register_signalsRegisters signals to the object it is attached to
